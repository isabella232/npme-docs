Updating the npm CLI

Updating the npm CLI

The npm CLI client ships with Node.js, but updates more often. Therefore, it is super important to update your npm regularly.

You can update npm by typing:

npm install npm -g

To learn more about what is new in new version of npm, you can read the CHANGELOG.
